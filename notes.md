# Yazoo - Moteur de recherche de media

## Features
 - Search Video (youtube/dailymotion)
 - Search Audio (bandcamp/soundcloud/mp3)
 - Search Images (stray files) .png .jpg .jpeg .gif .webm
 - Stats
 - Status
 - Blacklist domains
